# Desafio-DIO-Iphone
 Criação de um Diagrama UML simples e criação das interfaces e classes para o desafio DIO.

 O Diagrama UML possui 3 interfaces e 1 classe java.

 Sendo as Interfaces:
 -ReprodutorMusical
 -NavegadorInternet
 -AparelhoTelefonico

 e a Classe Iphone que irá implementar as três interfaces.
